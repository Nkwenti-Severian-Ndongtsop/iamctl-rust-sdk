# Contributing

Thanks for your interest in contributing to **iamctl-rust-sdk**!

## Ways to contribute

- Report bugs
- Suggest features
- Improve documentation
- Submit pull requests

## Development setup

### Rust

- Install Rust (stable)
- Build:

```bash
cargo build
```

- Test:

```bash
cargo test
```

### Docs (Docusaurus)

From `iamctl-docs/`:

```bash
npm ci
npm run start
```

## Pull request process

- Keep changes focused and small
- Add/update tests when applicable
- Update docs if you change public behavior
- Ensure CI passes

## Code style

- Prefer clear names and explicit error handling
- Avoid breaking changes without discussion

## Reporting issues

- Use GitHub Issues for bugs/feature requests
- Use **Security Policy** for vulnerabilities
